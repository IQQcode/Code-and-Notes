超级玛丽 童年的乐趣

超级玛丽，全名"超级马里奥兄弟"，是任天堂公司出品的著名横版游戏。最早在红白机上推出，有多款后续作品，迄今多个版本合共销量已突破4000万套。 其中的主角马里奥、路易、碧琪公主、奇诺比奥等等已成为任天堂的招牌人物。马里奥是站在游戏界顶峰的超人气多面角色。 马里奥靠吃蘑菇成长，特征是大鼻子、头戴帽子、身穿背带裤，还留着胡子。 与他的双胞胎兄弟路易基一起，长年担任任天堂的招牌角色。

大白 也有一颗超人的心

大白，是电影《超能陆战队》及动画《大白归来》中登场的虚拟人物，是一个体型胖嘟嘟的充气型智能机器人，因呆萌的外表和善良的本质获得大家的喜爱，被称为“萌神”、“守护性暖男”。大白是主角小宏的哥哥研发的“私人健康顾问”，也就是一个智能机器人，与其他机器人与众不同的是，它是一个充气机器人。

阿童木 童年小英雄阿童木

阿童木一直以为自己的名字是天马杜比，是大都会最出色的科学家天马博士的十三岁儿子。其实，他是一个最先进的机械人， 是博士依照其于实验室意外身亡的爱儿外貌及性格创造。由于阿童木无法满足博士的期望，因此惨遭博士遗弃。 阿童木不仅无法接受父亲不再爱自己，更要承受自己不是人类的可怕事实。

小黄人 呆萌又可爱

《小黄人》围绕小黄人们的“前格鲁时代”展开叙述，并特别追溯了小黄人的起源。小黄人总是忠诚地为主人提供各种服务，兢兢业业，毫无怨言，直到它们失手“害死”每一任主人。在痛苦地失去一个又一个主人后，被困在冰洞之内小黄人们愈发焦躁不安， 他们决定挺身而出，去到花花世界，再次寻找可以效劳的主人。

哆啦A梦 多少人童年的梦想

哆啦A梦，来自未来世界（22世纪）的蓝色猫型机器人，身上的四次元百宝袋装有许多道具，为改变大雄的命运而来。因为失去原生的耳朵（电视动画设定被真正的老鼠咬掉，《2112年哆啦A梦的诞生》设定被机械鼠咬掉）， 常被人以为是蓝色的狸猫而引起不满。弱点为害怕老鼠，喜欢吃铜锣烧饼。
